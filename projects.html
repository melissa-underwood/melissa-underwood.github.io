Front-End Prototype

Github

This is a front-end web application prototype for a fictional realty company (Oasis Realty Group) showcasing multiple properties.

Technologies Used

  • Initially designed on Figma
  • HTML, CSS, Javascript, React.JS

Featured Pages & Functionality

Property Showcase

  • Displays information about property (house or apartment)
  • Able to redirect to another page for an application to tour property, purchase the property, or general inquiry.

Careers

  • Displays current job openings
  • Form to apply for a position

Resident Portal

  • Three main hubs: forum (for apartments), submit maintenance request, view lease details
